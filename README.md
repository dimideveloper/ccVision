# ccVision — ccLeaf Landing Clone

Eine Nachbildung der [ccleaf.com](https://ccleaf.com) Marketing-Landingpage als React-App.

## Starten

```bash
npm install
npm run dev
```

Öffne [http://localhost:5173](http://localhost:5173).

## Seiten

| Route | Beschreibung |
|-------|----------------|
| `/` | Landingpage (Home) |
| `/pricing` | Pricing mit Vergleichstabelle |
| `/faq` | FAQ mit Kategorien & Accordion |
| `/login` | Login (E-Mail, Google, Passwort) |
| `/register` | Registrierung |

## Build

```bash
npm run build
npm run preview
```

## Enthaltene Sektionen

- Navigation mit Glass-Effekt
- Hero mit Animations-Karussell
- Statistiken & Creator-Marquee
- Feature-Karten (Web Editor)
- Drag & Drop + Editor-Kompatibilität
- Before/After-Slider
- Themes/Presets
- Testimonials & Trustpilot
- Pricing (Essential / Plus / Ultimate)

## Hinweis

Dies ist eine **Demo-Landingpage** im Stil von ccLeaf — kein vollständiger Editor oder Backend. Buttons verlinken auf Anker (`#try`, `#pricing`).
